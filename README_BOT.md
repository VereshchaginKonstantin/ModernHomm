# Простой Telegram Бот

Простое приложение Telegram бота на Python, который отвечает на все сообщения стандартной фразой из конфигурации.

## Возможности

- Отвечает на все текстовые сообщения стандартной фразой
- Все настройки вынесены в конфигурационный файл
- Поддержка команд `/start` и `/help`
- Логирование всех действий

## Установка

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Настройте конфигурацию

Отредактируйте файл `config.json`:

```json
{
  "telegram": {
    "bot_token": "YOUR_BOT_TOKEN_HERE",
    "parse_mode": "HTML"
  },
  "bot": {
    "default_response": "Спасибо за ваше сообщение! Я простой бот и отвечаю всем одинаково."
  }
}
```

#### Параметры конфигурации:

**telegram:**
- `bot_token` - токен вашего бота (получите у [@BotFather](https://t.me/botfather))
- `parse_mode` - режим форматирования текста (HTML, Markdown, или MarkdownV2)

**bot:**
- `default_response` - текст стандартного ответа на все сообщения

### 3. Получите токен бота

1. Откройте [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен
5. Вставьте токен в поле `bot_token` в файле `config.json`

## Запуск

```bash
python bot.py
```

или

```bash
python3 bot.py
```

## Использование

После запуска бот будет:
- Отвечать на команду `/start` приветственным сообщением
- Отвечать на команду `/help` справочной информацией
- Отвечать на все остальные текстовые сообщения стандартной фразой из конфига
